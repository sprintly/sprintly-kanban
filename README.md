# Sprintly 5 Columns

> Oauth2 Client App w/ Hapi, React, Flux, and Sprinlty-Data

## Setup

**Prerequisites**

* **Create an Oauth Client** through the Sprintly admin with the credentials
in `config/default.js`

* node >= 0.10 and npm

```bash
$ make install
```

```bash
$ node server
server started on port 3600
```

Browserify/Less watch task:

```bash
$ gulp
```

Run tests:

```bash
$ npm test
```
